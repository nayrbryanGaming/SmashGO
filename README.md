# SmashGo - Effortless Badminton Court Booking

**One-liner**: The ultimate local-first corporate badminton management platform with ELO matchmaking and real-time WhatsApp coordination.
**Live Demo**: [https://smashgo.vercel.app/](https://smashgo.vercel.app/)

## Problem
Traditional badminton court booking is often fragmented, requiring manual coordination across multiple platforms. Players struggle to find opponents of similar skill levels, and admins face challenges managing overlaps and payments without a unified system.

## Solution
SmashGo provides a streamlined, mobile-first web application that centralizes booking and matchmaking. By prioritizing a local-first architecture, the system remains 100% operational even without a stable internet connection, ensuring reliability for both players and admins.

## Why SmashGo?
- **Local-First Reliability**: Works 100% offline using high-performance local storage.
- **Smart Matchmaking**: An ELO-based algorithm ensures competitive and fair play partners.
- **WhatsApp Integration**: Core transaction system for instant confirmation and coordination.
- **Privacy Focused**: No complex logins or invasive tracking—just play.

## Architecture
- **Frontend**: Next.js 14 (App Router) with TypeScript and Tailwind CSS for a premium, high-performance UI.
- **State Management**: React Context + LocalStorage for robust offline capabilities.
- **Backend (Optional)**: Supabase infrastructure ready for production scale and real-time syncing.
- **Design**: Modern Dark Mode with Glassmorphism and Framer Motion animations.

## What's Built
- [x] **Smart Booking Engine**: Overlap-safe booking with instant WhatsApp redirection.
- [x] **ELO Matchmaking System**: Intelligent player pairing based on skill and availability.
- [x] **Global Leaderboard**: Competitive ranking system to track community performance.
- [x] **Localization**: 100% strict English and Indonesian language support.
- [x] **Mobile-First PWA**: High-contrast, accessibility-optimized interface for all devices.
